# Scheduling Guidelines

## Time Budget

A daily pet care plan should fit the owner's available minutes when possible. If the plan exceeds the time budget, the app should explain why important tasks were kept.

## Priority Order

High-priority tasks should be scheduled before medium- and low-priority tasks. Lower-priority activities can be skipped when time runs out, but the schedule should explain the tradeoff.

## Time Conflicts

Overlapping tasks should be reviewed before the schedule is final. If two activities for the same pet overlap, adjust start times or move the less urgent task.
